# Dotfiles

## Installation

Using wget:

```bash
zsh <(wget -qO- https://raw.githubusercontent.com/Tyrbok/dotfiles/HEAD/installer/install.sh)
```

Or using curl:

```bash
zsh <(curl -s https://raw.githubusercontent.com/Tyrbok/dotfiles/HEAD/installer/install.sh)
```
